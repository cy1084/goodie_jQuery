취약한 액세스 제어

이 취약점은 액세스 제어 및 권한 부여가 불충분하게 적용되어 공격자가 권한이 없는 기능이나 데이터에 액세스할 수 있는 경우 발생합니다.

인젝션 공격

이 공격의 범주에는 NoSQL, OS 명령, LDAP, SQL 인젝션 공격이 포함되며 다른 사용자가 보는 웹 페이지에 JavaScript와 같은 악의적인 클라이언트 측 스크립트를 주입하는 크로스 사이트 스크립팅(XSS)도 포함됩니다.

암호화 오류

안전하지 않은 키 관리로 인해 발생할 수 있습니다. 이 위험은 또한 취약한 키 또는 난수 생성이나 암호화 프로토콜의 결함으로 인한 정보 유출 때문에 발생할 수 있습니다.

보안 구성 오류

웹 애플리케이션 프레임워크, 플랫폼, 서버 또는 보안 제어의 보안을 강화하지 않으면 무단 액세스, 민감한 정보 노출 또는 기타 보안 취약점이 발생할 수 있습니다.
